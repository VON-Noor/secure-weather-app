# 🌦️ Secure Weather App

## 📌 Description
This is a full-stack weather application that allows users to search for real-time weather data by city.

The project uses a secure backend to protect the API key and prevent exposure of sensitive data.

---

## 🚀 Features
- Search weather by city
- Real-time weather data
- Secure API handling (API key hidden)
- Error handling for invalid input

---

## 🛠️ Technologies Used
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express

API:
- OpenWeather API

---

## 🔐 Security Features
- API key stored in environment variables (.env)
- .gitignore used to prevent sensitive data leaks
- Backend used as a proxy to secure API calls
